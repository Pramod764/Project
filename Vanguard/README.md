# Vanguard Tech Store

A premium, next-generation e-commerce application built with **FastAPI** (Python) and modern **HTML/CSS/JS**.

## Features
- **Modern Aesthetics**: Glassmorphism design with a dark layout and sleek micro-animations.
- **FastAPI Backend**: Robust and fast Python backend using FastAPI and SQLAlchemy.
- **User Authentication**: Secure signup and login flow.
- **Product Management**: Dynamically fetched product catalog.
- **Shopping Cart**: Real-time cart management with local storage persistence.
- **Responsive**: Fully optimized for mobile and desktop displays.

## Setup Instructions

### 1. Prerequisites
- Python 3.8+ installed on your system.

### 2. Install Dependencies
Open your terminal in the project root and run:
```bash
pip install -r requirements.txt
```

### 3. Run the Backend
Navigate to the `backend` folder and start the server:
```bash
python backend/main.py
```
Wait for the message: `INFO: Uvicorn running on http://127.0.0.1:8000`.

### 4. Launch the Frontend
Simply open `frontend/index.html` in any modern web browser (Chrome, Edge, etc.).

## Project Structure
- `backend/`: Python FastAPI code and database.
- `frontend/`: UI code (HTML, CSS, JS).
- `requirements.txt`: List of Python packages required.
